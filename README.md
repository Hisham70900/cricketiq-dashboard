# 🏏 Cricket Analytics Dashboard

An interactive cricket analytics dashboard built using **Python, Pandas, Plotly, and Streamlit**.

## Features

- 📊 Dashboard Overview
- 🏏 Batting Analysis
- 🎯 Bowling Analysis
- 🏆 Team Analysis
- 📈 Interactive Charts
- 📥 CSV Export
- 🔍 Player Search

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## Dataset

Ball-by-ball cricket dataset containing over 280,000 deliveries.
